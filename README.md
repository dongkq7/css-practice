# css-practice
# CSS推荐书写顺序：
## 1. 定位和布局
- position
- top/bottom/left/right
- z-index
- float
- flex（display: flex; flex-direction; justify-content;align-items...）
## 2. 展示方式和可见性
- display
- opacity
- visibility
## 3. 盒子模型
- width/height
- box-sizing
- margin/border/padding
- box-shadow
## 4. 字体、文本效果
- font-family/font-size/font-weight/font-style
- line-height
- text-align
## 5. 背景相关
- background
## 6. 其他
- transform/transition/overflow/white-space...
